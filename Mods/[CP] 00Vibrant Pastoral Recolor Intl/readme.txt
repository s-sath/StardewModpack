--- INSTALLATION ---
> install the latest versions of SMAPI (https://smapi.io/) & Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915).
> unzip this mod into your Mods folder.
> run SMAPI to generate the config.json, then close the game.
> adjust the config.json to your preferences.
> run SMAPI again and play the game.


--- CONFIG OPTIONS ---
AESTHETICS
> Green Fall: changes fall from orange to green. minor tweaks to other terrain features to match.
	- options: true, false
	- default: true
> Grass Type: determines the look of grass.
	- options: vanilla, medium, full, vanillaShort, mediumShort, fullShort
	- default: 
> Water Type: determines the opacity of water.
	- options: opqaue, semi, transparent
	- default: opaque
> Flowering Bushes: adds flowers to bushes in all seasons.
	- options: true, false
	- default: true
> Snow-covered Bushes: dusts bushes with snow in the winter.
	- options: true, false
	- default: true
> Summer Bushes Replacement: replaces the large succulent bushes in summer with default ones.
	- options: true, false
	- default: false
> Terrain Flowers: replaces most terrain flowers with original sprites.
	- options: true, false
	- default: true
> Holiday Overlays: adds seasonal holiday decorations to buildings.
	- options: true, false
	- default: true
> Iridium Oasis: colors the desert Oasis building an iridium purple, for that extra bit of pizzazz.
	- options: true, false
	- default: false
